# Smart Waste Classification using Transfer Learning

## Project Overview

This project aims to automate household waste classification using Computer Vision and Deep Learning. A pretrained ResNet50 model is fine-tuned to classify waste images into multiple recyclable categories such as plastic, paper, glass, metal, cardboard, and trash.

The project follows an end-to-end machine learning workflow including dataset exploration, preprocessing, transfer learning, model evaluation, deployment using Streamlit, Docker containerization, and cloud-ready architecture.

---

## Project Goals

- Learn Computer Vision fundamentals
- Understand Transfer Learning
- Build an end-to-end Deep Learning pipeline
- Deploy a production-ready application

## Phase 1 - Dataset Exploration
